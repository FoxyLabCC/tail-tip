# TailTip
TailTip 是一个开源的设备，带有生物电采集功能和运动检测，并支持BLE5.2，用于表面肌电的采集和运动检测。
或者说，它可以（也是预计的用途）是作为边缘传感器使用，受基于高超-盆底肌动作通过塞子采集压力的启发，TailTip采集盆底肌的活动，并计算出一个分数/进度。
DG-LAB 郊狼协议的兼容在开发当中。
# 使用
TailTip带有三个通道，通道一、二(Ch1, 2)和右腿驱动(RLD)，以2.5mm TRS（三段式）连接，定义如下
| 通道 | Sleeve | Ring | Tip |
| --- | --- | --- | --- |
| Ch1 | GND | - | + |
| Ch2 | GND | - | + |
| Rld | GND | / | + |
郊狼3.0的TS(双极)线缆是不兼容的，因为当双极2.5mm插入三极时，GND和IN-会被短路，并且这种线缆不包含屏蔽层。但你或许可以用两根TS线缆临时替代，两个电极连接到Tip并计算CH=Ch1-Ch2尽管这不被推荐。
RLD电极能极大地提高信号质量，所以有条件尽量使用。
一个Ch用于主要地采集信号，两个电极放置于会阴部位，另外一个Ch用于辅助，是可选的。
如果只需要一个通道，AFE芯片或许可以使用ADS1291来降低成本。
推荐采用屏蔽线缆，单端接地。
（开发中）如果同时使用郊狼等电击设备，信号可能会被严重干扰，请将波形输出接到App，或者（在以后的版本）经过精密电阻分压网络和缓冲，直接把输出电极分一路接到TailTip上的Chx或者H1。
